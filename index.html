System Requirements

Before launching Minecraft Java Edition, ensure your operating system has a valid Java Runtime Environment (JRE) mapped to your path variable. Our custom launcher supports both OpenJDK 17 and OpenJDK 21 architectures seamlessly.

How to Authenticate

The launcher utilizes secure Microsoft OAuth 2.0 protocols. When launching for the first time, a secure local window will prompt you to log into your Xbox Live profile. The launcher never stores your plain text password; instead, it saves an encrypted refresh token locally inside your Windows app data profile directory.

  • Multi-factor authentication (MFA) is fully supported natively.
  • Session tokens automatically expire and renew every 24 hours.
  • Offline mode can be enabled via the compiled settings configuration sheet.
